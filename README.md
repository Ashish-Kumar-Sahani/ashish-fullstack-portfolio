# 🚀 Ashish Full Stack Portfolio

A modern, production-ready **Full Stack Developer Portfolio & Portfolio CMS** built using the MERN ecosystem. This project combines a premium public portfolio with a secure admin dashboard, enabling complete content management without modifying source code.

🌐 **Live Demo:** https://ashish-fullstack-portfolio-pi.vercel.app/

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ashish-Kumar-Sahani/ashish-fullstack-portfolio.git
```

### Install Dependencies

Backend

```bash
cd Backend
npm install
```

Frontend

```bash
cd Frontend
npm install
```

---

## Environment Variables

### Backend

```env
PORT=
MONGO_URI=
JWT_SECRET=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

### Frontend

```env
VITE_API_URL=
```

Backend
